send\_\-stimulus 

\subsection{Description}\label{namespaceRTC_Description}
Send a stimulus to an FSM that realizes this interface.\subsection{Semantics}\label{namespaceRTC_Semantics}
If the stimulus corresponds to any outgoing transition of the current state, that transition shall be taken and the state shall change. Any FSM participants associated with the exit of the current state, the transition to the new state, or the entry to the new state shall be invoked. If the stimulus does not correspond to any such transition, this operation shall succeed but have no effect.

If the given execution context is a non-\/nil reference to a context in which this FSM participates, the transition shall be executed in that context. If the argument is nil, the FSM shall choose an EVENT\_\-DRIVEN context in which to execute the transition. If the argument is non-\/nil, but this FSM does not participate in the given context, this operation shall fail with $\ast$ ReturnCode\_\-t::BAD\_\-PARAMETER.\subsection{Constraints}\label{interfaceRTC_1_1LightweightRTObject_Constraints}
