\subsection{Detailed Description}
NamingService management abstract class. This is the abstract interface class for NamingServer management. Concrete management classes must implement the following pure virtual functions.
\begin{DoxyItemize}
\item \doxyref{bindObject()}{p.}{classRTC_1_1NamingBase_ac8b6c5670ae869ba80b9407169cc7928} : Bind the specified object to NamingService
\item \doxyref{unbindObject()}{p.}{classRTC_1_1NamingBase_a78700b13ec16ffadfc80c13461db3832} : Unbind the specified object from NamingService
\end{DoxyItemize}
