\subsection{Detailed Description}
PortProfile helper class. This class manages the PortProfile that is profiles of the \doxyref{RTC}{p.}{namespaceRTC}:Port. This is mainly used in \doxyref{PortBase}{p.}{classRTC_1_1PortBase} class.
