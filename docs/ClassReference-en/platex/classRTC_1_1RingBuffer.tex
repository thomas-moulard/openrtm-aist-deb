\subsection{Detailed Description}
\subsubsection*{template$<$class DataType$>$ class RTC::RingBuffer$<$ DataType $>$}

Ring buffer implementation class. This is the buffer implementation class with ring shaped buffer of specified length. If data is stored in the entire buffer, data from now on will be overwritten from old data one by one. Therefore, only the length of latest data is stored in the buffer.

Note: In the current implementation, only last stored data can be read from the buffer.

\item[{\em DataType}]Data type to store in the buffer\end{DoxyParams}
