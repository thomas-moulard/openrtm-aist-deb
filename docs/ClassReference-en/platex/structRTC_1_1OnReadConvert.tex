\subsection{Detailed Description}
\subsubsection*{template$<$class DataType$>$ class RTC::OnReadConvert$<$ DataType $>$}

Data convert callback abstract class on read(). This is the interface for callback invoked when data is done read() from the InPort/OutPort's buffer. The return value of this callback will be the return value of read().
