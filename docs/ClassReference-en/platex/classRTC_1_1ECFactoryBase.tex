\subsection{Detailed Description}
\doxyref{ECFactoryBase}{p.}{classRTC_1_1ECFactoryBase} abstract class. This is the abstruct Factory classes for ExecutionContext creation. Concrete classes for each ExecutionContext creation must implement the following pure virtual functions.

This class provides the following public interfaces.
\begin{DoxyItemize}
\item \doxyref{name()}{p.}{classRTC_1_1ECFactoryBase_a521d5c350bf08586034dc034c46f5c70} : Get names of the target ExecutionContext for creation
